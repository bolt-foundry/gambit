# Object action demo

- TypeScript root: `examples/object_action/root.deck.ts`
- Markdown root: `examples/object_action/root.deck.md`
- TS action returns a structured teammate object; MD action emits an inner dialog note about the lookup.
- Usage (TS): `gambit run examples/object_action/root.deck.ts --input '"Jordan"' --verbose`
- Usage (MD): `gambit run examples/object_action/root.deck.md --input '"Casey"' --verbose`
